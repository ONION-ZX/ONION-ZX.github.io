移动端调试方法总结

Posted on 2019-03-06 | Post modified: 2019-03-06 | In 技术 |

最近回归租车项目了，被问到选择做pc还是m，考虑了一分钟后选择了m，原因是以前教我写程序的一个老师说过，选择自己没做过的事情才能成长。虽然租车那边pc我也没咋做过 = = 但觉得m对我来说绝对是更新的领域，应该可以学到更多的东西。然后就是接下来学习移动端调试方法，在此做个总结。

IOS系统

方案一

运行环境：Mac + Safari浏览器 + iPhone

调试步骤：

  1. 数据线连接iPhone和Mac
  2. iPhone开启Web检查器（设置 -> Safari -> 高级 -> 开启Web检查器）
  3. iPhone使用Safari浏览器打开要调试的页面
  4. Mac打开Safari浏览器调试（菜单栏 -> 开发 -> iPhone设备名 -> 选择调试页面）
